Push notifications with service worker

We are using service worker notifications. If the app was added to home screen (installed), then notification can be displayed even when the app is inactive. To listen to events some changes need to be done to Angular 5 ngsw-worker.js. Check our upgraded file here ./good-ngsw-workerngsw-worker.js It gets copied automatically instead of ngsw-worker.js during build.
Notification can display an icon, title and content and some more details (see example in code).
It works only on Android for now, not on IOS.
